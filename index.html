Kết quả đạt được
  • Loại bỏ logic hard-code và if-else phức tạp theo type
  • Giảm coupling giữa các hệ thống — thay đổi 1 nơi không vỡ nơi khác
  • Thêm feature mới không cần sửa code cũ (Open/Closed principle)
  • Code rõ ràng và dễ maintain so với version monolithic ban đầu
